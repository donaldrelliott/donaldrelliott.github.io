Traveling Salesman Problem

C++/Python

Project done in conjunction with Noah Beach and John Carrabino. Goal was to find the most optimal routes for a given amount of coordinates/distances. John utilized the Ant Colony model (Python), Noah worked with Nearest Neighbor (Python), and I tackled it with the Christofides Algorithm utilizing a 2-OPT heuristic model (C++).
